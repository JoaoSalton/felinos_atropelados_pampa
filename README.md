# Pampa Roadkill Felines Map

Interactive map with roadkill records of felines from the Felinos do Pampa database, southern Brazil.

Records from 2024 to 2026. Each species has a colored marker. Filter by year or view all records together. A heatmap layer shows record density.

## Species

- Geoffroy's cat (Leopardus geoffroyi)
- Margay (Leopardus wiedii)
- Southern tiger cat (Leopardus guttulus)
- Jaguarundi (Herpailurus yagouaroundi)
- Pampas cat (Leopardus munoai)
- Ocelot (Leopardus pardalis)
- Puma (Puma concolor)
- Domestic cat (Felis catus)
